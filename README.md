# Gears N' Beans

This is a website for a company that sells coffee products and offers bike services. The user can view the products, services and learn more about the company. <em>This is not a real company.</em>

## Technologies Used:

VENM Stack project

- Vue.js (Vue 3)
- Express.js
- MongoDB
- Node.js
- Vue Router
- NPM
- React.js
- Javascript
- AJAX
- CSS3
- Bootstrap
- FontAwesome
- HTML5
- Heroku
